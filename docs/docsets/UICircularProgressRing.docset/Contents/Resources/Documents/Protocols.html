  • UICircularProgressRingDelegate

    This is the protocol declaration for the UICircularProgressRing delegate property

    Important

    When progress is done updating via UICircularProgressRing.setValue(_:), the finishedUpdatingProgress(forRing: UICircularProgressRing) will be called.

    The ring will be passed to the delegate in order to keep track of multiple ring updates if needed.
